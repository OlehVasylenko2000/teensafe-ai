# TeenSafe AI: Safeguard Assistant for Youth

**TeenSafe AI** is an open-source, AI-driven assistant designed to protect displaced teenagers and youth within the EU. 

## Project Overview
This project is developed as part of the **UKRAINE Safeguard Alliance MTÃœ** (PIC: 879207044) initiative. It uses a low-code architecture (n8n) combined with Large Language Models to provide real-time safety guidance for teenagers.

## Core Modules
- **Cyber-Shield:** AI-powered analysis of communication risks (bullying, grooming, scams).
- **Digital Lawyer:** EU children's rights navigator (In Development).
- **Emotional Buddy:** Mental health and adaptation support (In Development).

## Tech Stack
- **Engine:** n8n
- **Models:** Google Gemini / OpenAI GPT-4o
- **Speech-to-Text:** OpenAI Whisper
- **Interface:** Telegram Bot API
