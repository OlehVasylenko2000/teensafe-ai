# Contributing to TeenSafe AI

Thank you for your interest in contributing to **TeenSafe AI**. As this project is dedicated to protecting vulnerable youth, we maintain high standards for safety, privacy, and code quality.

## How You Can Help
You can contribute in several ways:
- **Reporting Bugs:** Create an issue describing the bug and how to reproduce it.
- **Suggesting Features:** Propose new modules for safety, legal aid, or emotional support.
- **Improving Workflows:** Optimize our n8n JSON exports for better performance or accuracy.
- **Documentation:** Help us improve technical guides or safety prompts.

## Contribution Process
1. **Discuss First:** For major changes, please open an issue first to discuss your proposal.
2. **Fork and Branch:** Fork the repository and create a descriptive branch name (e.g., `feat/new-safety-filter`).
3. **Commit Messages:** Use clear, concise commit messages in English.
4. **Pull Request (PR):** Submit a PR to the `main` branch with a detailed description of your changes.

## Safety and Ethics
Since our target audience is minors, all contributions must:
- Adhere to **Privacy-by-Design** principles.
- Ensure 100% **GDPR compliance** (never store personally identifiable information).
- Use non-judgmental and supportive language in AI prompts.
